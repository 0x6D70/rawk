# rawk
Rawk is an interpreted programming language written in Rust.

## Build and setup
You need a working rust installation in order to build rawk yourself.
Nothing more is needed, yet.
You can build the project using cargo's `cargo run`

## Syntax
The syntax which is going to be used is defined [here](syntax.md)

## Usage
`rawk foo.rk`
